We made a program that would count the pixels of detected markers by measuring the height of the left and right side and taking the average. If given distance from marker and height of marker it would then calculate f. Below images were taken using the program and capturing with dimensions (480,270). We will use the mean of 625, as we see it has a downwards trend, the further away from the camera, the object is. We believe this is because we are losing resolution the further away the object is. We therefore believe that to get the best overall estimates, we will use the f value most likely to be closest to the actual value, since the variance, when taking the sqrt of it, it will become the standard deviation sqrt(22) = 4.69, which we can see fits well that some are 620, and some are 630, which are approx +-5 from the mean.
